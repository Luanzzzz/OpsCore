<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| workspace nav -> usuario | links guiam decisao operacional entre dominios |
| CSS global -> workspaces existentes | estilos compartilhados podem afetar inbox, execucao e agenda |
| `/api/panorama` -> browser | payload agregado continua sendo atualizado no cliente apos ajustes de shell |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-04-01 | T | nav compartilhada | mitigate | manter `href` fixos para `/inbox`, `/execucao`, `/agenda`, `/panorama` e active state por `usePathname` |
| T-04-04-02 | D | `WorkspaceShell.refreshWorkspace` | mitigate | preservar uma chamada para `/api/panorama`, estados de erro e sem polling automatico |
| T-04-04-03 | T | `src/app/globals.css` | mitigate | usar classes `panorama-*` escopadas e preservar familias `workspace-*` existentes |
| T-04-04-04 | S | links de workspace | mitigate | nav apenas aponta para workspaces reais; nao apresenta IA/integracoes como areas ativas |
</threat_model>
