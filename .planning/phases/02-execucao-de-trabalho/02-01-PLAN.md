## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| queries tasks→store JSON | dados validados viram source of truth local e nao podem perder rastreabilidade |
| queries tasks→queries inbox | a origem do inbox entra no dominio de execucao e deve ser preservada sem adulteracao |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-02-02 | T | `src/db/queries/tasks.ts` | mitigate | persistir `TaskOriginSnapshot` com campos definidos no tipo compartilhado e nunca aceitar origem arbitraria via cliente |
